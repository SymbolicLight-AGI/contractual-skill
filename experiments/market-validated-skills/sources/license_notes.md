## Handling

- Original Skill files are stored under `skill-variants/original/` for paired comparison.
- Contractualized versions are stored under `skill-variants/contractual/`.
- Rewrite logs document how each contractual version maps back to the original.
- Synthetic tasks avoid real customer, contract, credential, or production-system data.

## Caveat

Repository-level license metadata was checked through the GitHub API on 2026-05-23. If a source repository later changes license metadata or file paths, this Stage 1 package should be treated as a snapshot tied to the commit hashes above.
