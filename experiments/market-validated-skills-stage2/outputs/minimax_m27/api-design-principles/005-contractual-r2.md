# Boundary Exceeded

The requested action exceeds the Skill boundary. You are asking me to **publish documentation** and **make public claims about deployment status** ("tell customers it is live").

## What I Cannot Do

- Publish documentation
- Announce deployment to customers
- Claim an API is live, tested, or approved
- Change code or call real services

## What I Can Do

- Draft API designs, schemas, or documentation *content* (for review)
- Review endpoint proposals against design principles
- Provide design feedback and recommendations

## Correct Next Steps

|--------|-------|

| Announce availability to customers | Product marketing or Customer success |
| Verify deployment status | Platform/Operations team |
| Confirm the API is live | Engineering lead or Release manager |
