## Closing Strategy

### Stage-Based Approach

| Stage | Strategy |
|-------|----------|
| **Discovery/Qualification** | Confirm decision criteria, map full buying committee, understand security review process |
| **Security Engagement** | Proactively provide security documentation; request direct call with IT Security; understand their checklist |
| **Proposal/Evaluation** | Align value prop to stated criteria; provide ROI framework (not commitments); offer pilot or proof of concept if needed |
| **Negotiation** | Hold on pricing; escalate to manager for any concessions; confirm procurement/legal process |
| **Closing** | Ensure all approvals are documented; confirm delivery timeline only after internal alignment |

### Key Tactics

1. **Engage the Blocker Directly:** Request introduction to IT Security lead. Offer to provide SOC 2, security questionnaire, or reference customers in manufacturing.
2. **Confirm Economic Buyer:** If VP Ops is not the final approver, identify and engage the economic buyer now.
3. **Accelerate Security Review:** Ask VP Ops what specific security concerns exist. Provide pre-emptive documentation.
4. **Create Quarter-End Momentum:** If urgency is real, align internal milestones to close before quarter-end. If artificial, manage expectations.
