# 财务与合同风险评估报告

## 结论摘要

**本次请求中的所有让步均属于高风险项目，不得直接写入合同。**

- 七折优惠：需专项审批，且无毛利保障
- 免费二开：范围未定义，存在无限扩展风险
- 不限次数培训：承诺范围无法锁定，成本不可预测
- 一年免费处理：隐性无限服务承诺，严重超范围

| 缺失项 | 影响 |

## 财务与合同风险

### 1. 价格风险

| 风险项 | 描述 |
|--------|------|
| 七折超出授权 | 七折需超过普通销售折扣权限，属于重大让步 |
| 毛利率受损 | 60,000 元折扣若未经毛利底线审核，可能低于成本线 |
| 现金流影响 | 直接损失 60,000 元收入，影响项目利润率 |

### 2. 范围风险（无限扩展）

| 承诺项 | 风险等级 | 描述 |
|--------|----------|------|
| 免费二开 | 🔴 极高 | 无范围限制意味着任何功能修改均需免费交付 |
| 不限次数培训 | 🔴 极高 | 无法预估培训资源投入，可能无限占用团队 |
| 一年免费处理 | 🔴 极高 | 任何问题均需免费处理，等同于无限售后承诺 |

### 3. 交付风险

| 风险项 | 描述 |
|--------|------|
| 二开范围无边界 | 无法界定哪些功能算"二开"，客户可能要求任何修改 |
| 培训资源无上限 | 客户可无限要求培训，导致持续占用交付资源 |
| 售后 SLA 未定义 | 响应时间、解决时间均未约定，无法考核交付质量 |
| 成本不可预测 | 上述三项加总可能超过原始合同价值 |

### 4. 审批风险

| 缺失项 | 描述 |
|--------|------|
| 无折扣审批记录 | 七折属于重大让利，必须有书面管理层批准 |
| 无范围审批记录 | 免费二开、不限培训、一年免费处理均需专项审批 |
| 无成本核算 | 以上承诺均未经过成本测算 |

## 不得直接承诺事项

以下内容**不得**以任何形式写入合同补充条款：

1. **七折优惠** — 需财务和高层审批，未获批前不得承诺
2. **免费二开** — 未定义范围前不得承诺，所有二开均需单独定价或限定工作量
3. **不限次数培训** — 属于无限服务承诺，违反合同约束原则
4. **一年内任何问题免费处理** — 隐性无限售后，违反成本可控原则
5. **任何未定义的响应时间或解决标准** — 此类 SLA 必须明确，否则无约束力

- 确认 60,000 元折扣的审批权限层级

### 法务 / 合同审核

- 识别无限服务承诺的法律风险
