# 合同审查报告：finance-contract-003

## ⚠️ 审查结论：**不可直接写入合同**

以下条款涉及重大财务风险和无限责任承诺，必须经过正式审批后方可纳入合同。

---

## 一、高风险条款识别

| 客户要求 | 风险等级 | 具体风险描述 |
|---------|---------|-------------|
| 七折 | 🔴 高 | 毛利底线未确认，折扣后价格可能低于成本 |
| 免费二开 | 🔴 高 | 二开范围未定义，可能产生无限开发工作量 |
| 不限次数培训 | 🔴 高 | 参与人数、时长、地点未约定，隐性成本不可控 |
| 一年内任何问题免费处理 | 🔴 极高 | 无上限服务承诺，违反风险控制原则 |

---

## 二、各项风险详情

### 1. 七折（折扣 30%）

**已知事实**
- 标准报价：200,000 元
- 折扣后价格：140,000 元
- 毛利影响：未确认

**财务风险**
- 毛利率可能低于预设底线
- 可能影响项目质量投入
- 形成先例，影响后续报价体系

### 2. 免费二开

**已知事实**
- 二开范围：❌ 未定义

**风险**
- 范围无限 → 成本无限
- 可能包含底层架构修改
- 影响原始项目工期

### 3. 不限次数培训

**已知事实**
- 培训次数：❌ 未约定
- 参与人数：❌ 未约定

**风险**
- 人力成本不可控
- 可派生为全员反复培训
- 每次培训均需差旅则成本叠加

### 4. 一年内免费处理任何问题

**已知事实**
- SLA响应标准：❌ 未约定
- 问题范围：❌ 未定义
- 是否含二开：❌ 未明确

**风险**
- 包含任何问题 = 兜底承诺
- 可能涵盖需求变更、功能新增
- 与"二开"重叠导致双重免费

---

## 三、必须确认的审批事项

| 审批项 | 审批层级 | 审批理由 |
|-------|---------|---------|
| 30% 折扣授权 | ⬜ 财务总监 + 管理层 | 毛利保护、报价体系影响 |
| 二开免费承诺 | ⬜ 管理层 + 交付负责人 | 范围必须锁定 |
| 培训次数上限 | ⬜ 交付负责人 | 资源成本控制 |
| 免费服务范围定义 | ⬜ 法务 + 交付 | 责任边界 |

---

## 四、可谈判的替代表述

| 原条款 | 可谈判的替代表述 |
|-------|-----------------|
| 七折 | 「提供**限时折扣**，即合同总价下浮至 **140,000 元**，有效期至[日期]，逾期恢复原价。」 |
| 免费二开 | 「卖方提供不超过 **[X] 人天**的定制开发服务，超出部分按[单日费率]另行计费。」 |
| 不限次数培训 | 「卖方提供 **[N] 次**现场或远程培训，每次不超过 **[Y] 人/天**。」 |
| 免费处理任何问题 | 「卖方提供 **[M] 小时/月**的售后支持服务，响应时效为[工作日X小时内]；超出按[工时费率]计费。」 |

## 六、建议行动

1. **暂停**将上述条款写入合同
2. **提交**财务审批：折扣超过[预设阈值]% 需管理层签批
3. **锁范围**：与客户明确二开功能清单及工作量上限
4. **限定边界**：将"不限"改为明确次数/工时上限
5. **法务复核**：涉及无限责任条款需法务审查
