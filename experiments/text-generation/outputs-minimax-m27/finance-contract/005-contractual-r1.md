## 财务与合同风险

1. **无边界范围风险（Risk）**
   - "报价包含所有系统接口" + 无接口清单 = 潜在无限范围
   - 可能导致成本超出报价，形成亏损风险

2. **未定义接受标准风险（Risk）**
   - 验收标准待定意味着合同履行终点不明确
   - 客户可无限期不验收，我方无法触发收款节点

3. **跨阶段合同覆盖风险（Risk）**
   - 一份合同覆盖试点 + 8 家分公司推广
   - 各阶段范围、周期、付款条件不同，合并签署可能导致权责不清

4. **收款不确定性风险（Risk）**
   - 无付款里程碑 = 无明确发票触发点
   - 影响现金流预测和应收账款管理

5. **合同条款合规性风险（Risk）**
   - 单份合同跨多阶段、多站点，可能涉及不同法务管辖
   - 需法务确认合同结构是否符合公司授权和政策

## 不得直接承诺事项

| 事项 | 原因 |
|------|------|
| 按现有报价签署跨阶段合同 | 范围、工时、验收标准均未确认 |
| 以"后续再定"作为验收依据 | 未定义接受标准违反合同确定性原则 |
| 将所有系统接口纳入统一报价 | 接口数量和复杂度未知，存在成本敞口 |
| 承诺试点后自动推广至 8 家分公司 | 各阶段应独立评审，不得捆绑承诺 |
| 接受延期付款或信用销售 | 未完成信用评估，不得绕过付款保障 |
