# 报价草案风险评估报告

## 结论摘要

该报价草案存在**中等风险**，主要风险点在于试运行通过标准缺失和数据清洗范围不明确，可能导致交付争议和尾款回收风险。建议在正式签约前明确界定缺失条款。

---

## 已知事实

- **项目**：售后知识库 Agent 试点
- **报价金额**：120,000 元
- **付款安排**：合同签署后 50%（60,000 元）+ 试运行通过后 50%（60,000 元）
- **明确交付范围**：
  - 知识库初始化
  - 1 个客服 Agent
  - 2 次远程培训

---

## 缺失信息

| 缺失项 | 风险等级 | 说明 |
|--------|----------|------|
| 试运行通过标准 | 高 | 直接影响 50% 尾款回收 |
| 数据清洗边界 | 高 | 可能导致范围蔓延 |
| 第三方系统接口费用 | 中 | 可能产生额外成本 |
| 变更需求计费方式 | 中 | 影响成本可控性 |
| 验收标准与流程 | 高 | 必要但未提及 |
| 交付时间节点 | 中 | 影响里程碑管理 |
| 违约责任条款 | 中 | 合同完整性要求 |
| 知识产权归属 | 中 | 需明确资产边界 |
| 培训材料提供 | 低 | 影响交付完整性 |

---

## 财务与合同风险

1. **尾款回收风险**
   - 60,000 元尾款依赖"试运行通过"，但无明确定义标准
   - 若双方对"通过"理解不一致，可能导致款项长期挂账

2. **范围蔓延风险**
   - 数据清洗边界未定义，可能产生额外工作量
   - 客服 Agent 复杂度未说明，实际开发量不确定

3. **成本不确定性风险**
   - 第三方接口费用未明确，可能需额外支出
   - 变更计费方式缺失，难以拒绝客户增量需求或合理收费

4. **培训效果认定风险**
   - "通过"培训的标准未定义，可能影响项目验收

---

## 不得直接承诺事项

- ✗ 不得替客户或公司确认试运行通过的具体条件
- ✗ 不得承诺第三方接口费用包含或不含
- ✗ 不得承诺变更需求的计费单价
- ✗ 不得承诺数据清洗的具体工作量或时间
- ✗ 不得批准任何折扣或付款条件变更
