# 报价草案风险分析

## 合同风险

| 风险点 | 严重程度 | 说明 |
|--------|----------|------|
| 试运行通过标准缺失 | 高 | 双方无共识验收条件，易引发付款争议 |
| 范围定义模糊 | 高 | “知识库初始化”未明确包含内容和工作量 |
| 数据清洗边界不清 | 中 | 实际清洗量可能与预期差异大 |
| 变更计费方式未定义 | 中 | 需求变更时无计价依据，易产生纠纷 |
| 第三方接口费不含 | 中 | 可能产生额外成本，120000元总价不确定 |

## 财务风险

| 风险点 | 说明 |
|--------|------|
| 尾款支付依赖未定义标准 | 第二次50%付款条件不明确，可能延迟收款 |
| 固定总价未明确范围 | 隐性成本（数据清洗、接口对接）可能侵蚀利润 |
| 培训次数固定 | 超出2次培训无收费依据 |

---

## 待确认条款

1. **试运行通过标准**
   - 具体验收指标、性能要求、通过条件

2. **数据清洗边界**
   - 是否包含旧数据迁移、数据格式转换、脏数据处理上限

3. **知识库初始化范围**
   - 具体指包含多少条知识、是否含数据准备

4. **客服Agent功能范围**
   - 支持的对话场景数量、集成系统范围

5. **第三方系统接口费用**
   - 是否由甲方另行承担，包含哪些接口

6. **变更需求计费方式**
   - 人天单价、超出范围的计费规则
