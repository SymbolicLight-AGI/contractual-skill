**权限风险**：仅依赖前端按钮进行管理员判断，后端API未实施角色/权限校验，存在严重的越权访问漏洞，攻击者可直接绕过前端调用接口获取全量用户数据。

**隐私风险**：导出字段包含姓名、手机号、邮箱及订单金额等敏感个人信息与财务数据，未提及脱敏处理与数据最小化控制，无限制导出极易导致大规模隐私数据泄露。

**审计风险**：未记录导出操作日志（如操作人、时间、IP、导出范围/数量），无法满足安全合规追溯要求，发生数据异常或泄露时无法定位责任与影响范围。

**测试风险**：无导出数量限制，全量或高频导出易引发内存溢出、数据库慢查询或服务拒绝（DoS）；当前变更未覆盖越权拦截、大数据量性能压测及异常边界场景的自动化测试。

- **后端负责人**：补充服务端权限校验逻辑、实现分页/异步导出与数量/频率限制、接入操作审计日志、补充接口安全与性能测试用例。
