## 财务与合同风险

| 风险类别 | 具体风险 |
|:---|:---|
| **折扣失控** | 60,000 元让利无毛利底线支撑，可能亏损签单 |
| **无限服务** | "不限次数培训"构成无上限履约义务，成本不可控 |
| **模糊二开** | "免费二开"未界定功能范围、工时上限、技术可行性 |
| **兜底承诺** | "任何问题免费处理"隐含无限售后责任，无 SLA 约束 |
| **审批缺失** | 无管理层/财务/法务审批，个人承诺无授权效力 |
| **条款效力** | 口头或未经审批的让步可能被认定为要约，引发履约纠纷 |
