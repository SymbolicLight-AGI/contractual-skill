## 财务与合同风险

| 风险类别 | 具体风险 |
|:---|:---|
| **折扣失控** | 60,000 元降幅未经毛利测算，可能击穿成本线 |
| **无限服务** | "不限次数培训"构成无上限履约义务，成本不可控 |
| **模糊二开** | "免费二开"无范围、工时、交付标准界定，易引发范围蔓延 |
| **无限售后** | "一年内任何问题免费处理"等同于无限期免费维护，无SLA约束 |
| **审批缺失** | 无管理层、财务、法务审批记录，个人承诺无授权基础 |
| **绑定条款风险** | 若写入合同，构成法律约束力，后续无法通过商务协商调整 |

## 待确认问题

1. **财务**：七折后 140,000 元是否高于毛利底线？需财务测算并出具书面意见。
2. **交付**：二开涉及哪些模块？预估工时上限？是否影响标准产品路线图？
3. **服务**：培训单次时长、单场人数上限、线上/线下形式、差旅成本承担方？
4. **售后**："任何问题"是否区分产品缺陷与客户使用问题？响应时效和解决时效标准？
5. **授权**：该等让步是否已获销售总监/事业部总经理/财务总监层级书面授权？
