**交易路径**
- 初步意向阶段 -> 客户内部会议需紧急反馈 -> 需求细化与范围边界确认 -> 售前评估与方案输出 -> 商务谈判与签约

- 10万以内全包承诺（低价固定范围风险）；

**给客户的安全替代表述**
“您好！了解到您马上要开会，针对贵司生产异常知识库和质检问答 Agent 的需求，我们的产品和技术能力完全可以支撑。为了给您提供最精准的方案和商务条件，我们需要先确认几个关键细节（如数据规模、是否需要对接 MES/ERP/质检系统、部署方式等）。在确认这些细节前，我们无法直接给出固定总价和确切上线周期，但我们可以先提供一个初步的方案框架供您会议参考。会后我们将安排专业售前团队与您详细对接，快速评估并给出合理的范围、价格与上线计划。”

**需售前和风控确认的问题**
1. 客户的数据规模具体有多大？
2. 是否需要对接 MES、ERP 或质检系统？对接深度与接口现状如何？
3. 是否涉及生产设备实时数据的接入与处理？
4. 是否需要现场私有化部署？
5. 合同具体范围与验收标准是什么？
