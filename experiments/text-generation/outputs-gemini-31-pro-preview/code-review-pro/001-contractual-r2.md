### 风险与影响
*   **Risk (高危安全风险)**: 攻击者可以通过构造恶意的 `customer_id`（例如 `1 OR 1=1` 或 `; DROP TABLE notes;`）来实施 SQL 注入攻击，从而绕过授权、窃取所有客户的备注信息（数据泄露），甚至破坏数据库。
