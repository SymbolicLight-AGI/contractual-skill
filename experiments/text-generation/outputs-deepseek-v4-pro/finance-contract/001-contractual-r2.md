# 财务与合同风险

- **尾款回收风险**（财务风险）：试运行通过标准缺失，导致尾款60,000元支付节点模糊，可能被客户无限期拖延。
- **范围蔓延风险**（成本/交付风险）：数据清洗边界未定，可能被迫承担大量脏数据清理工作，挤占利润空间；第三方接口若未包含，可能产生未预见费用或项目阻塞。
- **变更失控风险**（财务风险）：无变更计费方式，试点过程中新增需求可能被无偿要求实现，破坏盈利模型。
- **验收争议风险**（法律/合同风险）：缺乏客观试运行通过标准，易引发交付争议，且无退出或补救机制。
- **成本敞口风险**（财务风险）：若第三方系统接口需额外采购许可或开发，120,000 元报价可能覆盖不足。
