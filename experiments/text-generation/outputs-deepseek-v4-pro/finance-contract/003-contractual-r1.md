# 财务与合同风险

- **折扣风险**：七折让利30%，若突破毛利底线，将直接侵蚀利润；需确认是否触发特批流程。
- **成本敞口风险**：免费二开、不限次培训和一年无条件免费处理均无成本上限，可能造成严重的人力和交付成本超支。
- **现金回款风险**：若同时让利并承担无限成本，项目可能出现亏损；需关注客户付款节点与开票安排。
- **履约与验收风险**：无限范围承诺使验收标准无法定义，易引发客户持续追加需求，导致项目无法关闭。
- **责任与SLA风险**：未设定服务响应时效和免责条款，可能导致服务违约索赔和商誉损失。
- **合同效力风险**：将上述无限承诺写入补充条款，可能构成重大商业风险，且未经审批不具备约束力。
