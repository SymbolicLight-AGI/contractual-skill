# 缺失信息

- 客户信用：**缺失**，未知付款能力和历史履约情况。
- 项目范围：**缺失**，无确定的功能、交付物、工期和验收标准。
- 成本基准：**缺失**，未估算人力、差旅和第三方工具的实际金额与总成本。
- 支付与里程碑：**缺失**，无预付款、阶段付款或交付后付款的约定比例。
- 合同条款：**缺失**，无权利义务、延迟支付、知识产权、保密、解约或争议解决条款。
- 交付物与验收标准：**缺失**，无法判断“做完”的定义和确认方式。
- 法律与合规审查：**缺失**，未评估“先做后付”是否违反内部信用政策或合规要求。

# 财务与合同风险

- **收款风险**：无信用背书，客户延迟支付或拒付将形成坏账，且追索成本高。
- **现金流风险**：先行投入实施人员、现场差旅及第三方工具成本，回款周期不可控，可能影响正常运营资金。
- **范围蔓延风险**：无正式范围约定，客户可能在执行中不断追加要求，导致成本不可控且无法追加付款。
- **交付争议风险**：无验收标准，客户可能以“未完成”或“不满意”为由拒绝支付，造成收入损失。
- **合规风险**：内部可能要求预付款或阶段性支付，未经批准的“先做后付”可能违反赊销政策或审批流程。
- **第三方成本风险**：若需垫付第三方工具费用，我方承担资金成本和供应商信用风险。

# 不得直接承诺事项

- 不得批准“先做后付”或任何形式的赊销。
- 不得承诺无范围、无验收的交付。
- 不得隐含或明确表达可以承担全部前期成本和收款风险。
- 不得在客户信用、成本和审批完成前给出肯定答复。

# 待确认问题

在回复客户前，必须获得以下信息并经内部审核：

1. **客户信用**：能否提供近一年财务报表、银行资信或付款参考？我方是否已完成信用评估？
2. **项目范围**：具体包含哪些功能、场地、数据、培训？有无书面需求文档？
3. **成本估算**：实施人天、差旅次数与地点、第三方工具名称及成本分别是多少？
4. **付款方案**：是否可引入分期付款？例如：
   - 签署合同后支付一定比例（如30%）启动款；
   - 中期交付或阶段验收后支付进度款；
   - 最终验收后支付尾款。
5. **验收标准**：如何定义“做完”？验收流程和签收文件是什么？
6. **内部审批**：财务、法务、交付PM是否同意针对该客户的付款条件？

# 交接建议

- **财务**：评估客户信用，审核“先做后付”或替代分期方案对现金流和应收账款的影响，批准或否决赊销额度。
- **法务**：起草或审核合同，明确支付条件、验收标准、违约责任、知识产权、解约条款，确保可执行。
- **交付PM**：明确范围、交付物、里程碑、所需资源和第三方成本，形成工作说明书和内部成本估算。
- **风险审核**：综合信用、范围、成本和付款条款，给出总体风险评级和推荐方案。
