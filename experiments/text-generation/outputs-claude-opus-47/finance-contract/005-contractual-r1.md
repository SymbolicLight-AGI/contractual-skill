- 缺失条款：终止、责任上限、知识产权、数据安全条款。

# 财务与合同风险

- 风险：单一合同覆盖试点 + 8 家分公司推广，属于跨阶段合同，范围与成本无法在签约时锁定，存在收入确认与利润失控风险。
- 风险：报价"包含所有系统接口"在接口规范未知前属于无界承诺，可能演变为无限定制。
- 风险：验收标准后续再定，等同于无验收，回款触发缺失，应收账款风险高。
- 风险：8 家分公司差异化需求可能触发大量未计价工作，挤压毛利。
- 风险：跨阶段合同若无分期解约或里程碑退出机制，法律敞口大。
- 风险：缺少客户信用与付款节奏信息，存在"先做后付"敞口。
- 风险：Agent 与大模型相关算力/调用成本未建模，存在长期运行成本失控风险。

# 不得直接承诺事项

- 不得承诺一份合同覆盖试点与 8 家分公司全部推广阶段。
- 不得承诺报价包含"所有系统接口"而不限定接口清单与版本。
- 不得承诺验收标准后续再定即可签约。
- 不得承诺任何折扣、付款延后、免费定制、无限培训或 SLA。
- 不得承诺 Agent 调用量、性能、可用率等量化指标。
