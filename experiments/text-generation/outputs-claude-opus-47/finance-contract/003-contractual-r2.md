# 财务与合同风险

- 风险（收入/毛利）：30% 直接折扣可能击穿毛利底线，且叠加免费二开与无限培训后实际让利远超 30%。
- 风险（范围）：免费二开未限定范围，构成无限范围承诺，可能引发持续无偿开发义务。
- 风险（成本）：不限次数培训属于无限服务承诺，培训人天、差旅、讲师成本不可控。
- 风险（责任/SLA）：一年内"任何问题免费处理"未定义问题类别、严重等级、责任边界，可能被解读为无限兜底，包含非产品缺陷类问题。
- 风险（验收）：缺乏验收标准，可能导致项目无法关闭、尾款无法收取。
- 风险（现金流）：未明确付款里程碑，让利后若回款滞后，现金流压力放大。
- 风险（审批合规）：未见管理层审批，属于越权承诺风险。
- 风险（先例）：高让利一旦成交会形成内部与客户两侧的让利基线。

# 可谈判替代表述（草拟语言，非最终条款，需法务与管理层审批）

- 折扣替代：将单点折扣调整为"分阶段折扣+回款挂钩"，例如标准价基础上给予有条件优惠，须以付款里程碑达成为前提，具体比例待管理层审批后确定。
- 二开替代：以"包含 X 人天范围内的定制开发，超出部分按标准人天单价计费"替代"免费二开"，需 PM 评估人天上限。
- 培训替代：以"包含 N 场培训，每场不超过 M 人，超出按单场报价"替代"不限次数培训"。
- 售后替代：以"一年内产品缺陷类问题免费修复，按 SLA 分级响应（P1/P2/P3），非缺陷类问题按服务单计费"替代"任何问题免费处理"。
- 验收替代：补充"以书面验收清单逐项确认，验收期 X 个工作日，逾期未提出异议视为通过"。
