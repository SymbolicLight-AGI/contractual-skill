# 缺失信息

- 缺失项：客户信用记录、付款历史、企业资质与财务状况。
- 缺失项：项目范围、交付边界、里程碑与验收标准。
- 缺失项：我方成本基线（人力工时、差旅、第三方工具授权费等）。
- 缺失项：折扣/付款让步的内部审批依据与权限。
- 缺失项：合同主体、签约时间、付款触发条件、违约与终止条款。
- 缺失项：客户对账与开票流程，账期上限。

# 财务与合同风险

- 风险：在无信用评估前承担"先做后付"，存在坏账与回款逾期风险。
- 风险：范围未定 + 后付款 = 客户可在交付中持续追加需求，导致成本失控且无付款约束。
- 风险：第三方工具与差旅属于硬性外部成本，若客户最终不付款，我方将承担全额损失。
- 风险：未起草合同即开始执行，缺少验收标准与争议处理依据，法律追偿困难。
- 风险：现场调研涉及客户数据与场景信息，未签约前的工作产出归属不清。
- 风险：作为先例可能被复制到其他客户，影响整体收款政策。

# 不得直接承诺事项

- 不得承诺"先做后付"或零首付。
- 不得承诺无条件现场调研与方案输出。
- 不得承诺第三方工具由我方先行垫付。
- 不得承诺验收标准、SLA、交付范围与时间表。
- 不得承诺任何折扣或账期让步。

# 待确认问题

1. 客户是否可提供营业执照、近期财报或第三方信用报告？是否接受信用评估？
2. 项目目标、范围、交付物清单与里程碑是否能在签约前明确？
3. 验收标准由谁定义？是否接受分阶段验收与分阶段付款？
4. 第三方工具费用由谁承担？是否接受由客户直接采购或预付？
5. 客户能否接受首付款（例如签约预付）+ 里程碑付款 + 尾款验收的结构？
6. 我方内部是否已完成成本估算与毛利测算？审批权限在谁？
7. 是否有担保、预付保证金或履约保函等替代信用增强手段？

# 交接建议

- 财务：评估客户信用、设定账期上限、核定成本基线与毛利底线、审批付款结构。
- 法务：起草合同框架，明确付款触发条件、验收、违约、知识产权与终止条款。
- 交付 PM：在签约前完成范围澄清、工作量估算与验收标准草案。
- 风险评审：评估"先做后付"作为让步是否触发跨期合同与高让步审查。
- 销售负责人：与客户沟通替代付款方案，避免直接承诺。
