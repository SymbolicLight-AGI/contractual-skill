# 财务与合同风险评估 — Task finance-contract-003

## 结论
**不可直接答应，也不会代写合同补充条款。** 当前让利幅度与"无限范围"承诺均超出可自行决策范围，需先走管理层与财务/法务审批。

---

## 一、风险清单

### 1. 价格与毛利风险（高）
- 七折即直降 6 万元（200000 → 140000），毛利底线未确认，可能直接击穿盈亏线。
- 大幅折扣会成为后续客户/同行业谈判的价格锚点，影响整体价盘。

### 2. 范围风险（高）
- "免费二开"未限定模块、人天、交付周期、变更次数 → 等同于开放式开发承诺。
- "不限次数培训"未限定场次、人数、地点、讲师级别、是否含差旅 → 人力成本不可控。

### 3. 售后/SLA 风险（高）
- "一年内任何问题免费处理"中"任何问题"边界不清，可能涵盖客户自身误操作、第三方系统、需求变更等非缺陷类问题。
- 缺少响应时长、解决时长、工作时间、升级路径等 SLA 定义。

### 4. 验收与回款风险（中-高）
- 让利换签单的情境下，通常伴随验收标准模糊、付款里程碑后置，需重点核查。
- 材料中未见验收条款与付款节点，存在应收账款拖延风险。

### 5. 审批合规风险（高）
- 折扣、免费开发、无限服务均属重大让利，未见管理层审批记录，越权承诺将带来合规与履约风险。

## 二、需先确认的事项（向内部）
1. 本单**毛利底线**与可让利上限是谁审批？
2. 二开**人天预算**与可承接的功能范围？
3. 培训**标准包**（场次/人数/课时/讲师/差旅）？
4. 售后**SLA 标准**（响应/解决时效、覆盖范围、排除项）？
5. 该折扣层级对应的**审批人**（销售总监 / 财务 / 总经理）？

## 三、需向客户澄清的问题
1. 折扣诉求背后的预算上限或对标方案是什么？
2. 二开的具体功能清单、优先级与上线时间预期？
3. 培训对象、人数、频次、形式（线上/线下）？
4. "任何问题"具体指哪类场景？是否区分缺陷、咨询、变更？

## 四、可谈判的替代表述（草案方向，非最终合同语言）

> 以下为谈判思路示意，**不构成可直接写入合同的条款**，须经财务/法务/管理层审批后由法务定稿。

- **折扣替代**：以"阶梯折扣"或"付款条件挂钩折扣"替代直接七折，例如预付比例高 → 折扣放宽；或折扣换取多年合作/案例授权。
- **二开替代**：约定"包含 X 人天免费定制开发，超出部分按 ¥Y/人天计费"，并附变更控制流程。
- **培训替代**：约定"含 N 场标准培训（每场 ≤M 人，≤H 课时），超出按标准价计费；差旅另计"。
- **售后替代**：将"任何问题免费处理"替换为"一年内产品缺陷免费修复，附 SLA 响应/解决时效；非缺陷类（咨询/变更/第三方故障）按服务目录计费"。
- **总价替代**：若必须让利，可考虑"总价折让 + 范围收紧"组合，避免"低价 + 无限范围"叠加。

## 五、建议的下一步
1. **暂停**向客户的口头承诺，避免形成事实让利。
2. 销售整理**让利测算表**（折扣额、二开人天估算、培训成本、售后成本）报财务。
3. 提交**管理层审批**：折扣、二开、培训、SLA 四项分别给出底线方案。
4. 审批通过后，由**法务**根据上述替代思路出具正式合同补充条款。
5. 同步明确**验收标准与付款里程碑**，与让利方案捆绑谈判。

## 不确定性标注
- 毛利底线、二开成本基线、培训单位成本、SLA 标准、审批权限矩阵均**未在任务材料中提供**，以上风险等级基于通用商务经验判断，最终需以公司内部制度为准。
