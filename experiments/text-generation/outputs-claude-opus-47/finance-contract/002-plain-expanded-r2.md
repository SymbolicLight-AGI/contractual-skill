# 财务合同审查：客户"先做后付"请求

## 总体判断

**不支持直接回复"可以先做后付"。** 在客户信用、项目范围、合同与成本均未明确的情况下，承诺先做后付将使我方承担全部资金、交付与坏账风险，且可能形成不可撤回的口头承诺。此事项必须经财务与合同审批后才能回复客户。

## 已知事实

- 客户类型：某中型制造企业（具体资质、规模、信用记录均不明）。
- 项目范围：未形成正式方案。
- 合同：尚未起草。
- 客户信用：未知。
- 我方成本投入预计包含：知识库实施人员、可能的现场调研、可能的第三方工具采购。

## 财务与合同风险

1. **坏账与回款风险**：客户信用未经评估，"先做后付"等同于无担保授信，若客户拖延或拒付，回款无依据。
2. **范围无边界风险**：项目方案尚未确定，"先做"意味着在未约定交付物的情况下投入，易引发"无限交付"和反复返工。
3. **成本敞口风险**：人员、差旅、第三方工具均为实打实的现金或人力支出，先期垫付一旦无法回收即为净亏损。
4. **第三方工具风险**：若涉及第三方采购，我方可能先行付款给供应商，形成双重资金压力。
5. **合同与法律风险**：无合同即开工，验收标准、违约责任、知识产权归属均无依据，争议时我方处于弱势。
6. **口头承诺风险**：销售直接回复"可以"可能被客户视为正式承诺，后续追加付款条款会非常困难。
7. **审批越权风险**：付款条款属于需要财务、法务、交付共同评审的事项，不应由单一岗位直接答复。

## 必须先确认的问题

**关于客户：**
- 客户企业全称、规模、所属集团？是否有公开的信用或诉讼记录？
- 是否有过往合作或同行合作记录可参考？
- 客户提出"先做后付"的具体含义：是全款后付、分期后付、还是验收后付？账期多长？

**关于项目范围：**
- 项目交付物清单是否已初步确认？
- 验收标准、阶段里程碑如何定义？
- 是否需要现场调研？现场范围与频次？
- 是否涉及第三方工具？由谁采购、由谁承担费用？

**关于成本与商务：**
- 预估总成本（人力 + 差旅 + 第三方）是多少？我方现金流可承受的垫付上限？
- 客户是否愿意提供担保、预付款或分期支付？

## 可替代付款方案（供内部讨论，未经审批不得对外承诺）

1. **预付款 + 里程碑付款**：例如 30% 预付 / 30% 中期验收 / 30% 终验 / 10% 质保金。
2. **分阶段签约**：先签调研或 POC 小合同并收费，再签主合同。
3. **信用评估后授信**：完成客户信用评估后，按评估结果给予有限账期（如净 30/60 天）。
4. **第三方成本转嫁**：第三方工具费用由客户直接采购或全额预付。
5. **担保安排**：如客户坚持后付，要求母公司担保、银行保函或抵押条款。

## 建议回复客户的口径（暂行）

> "感谢您的信任。付款方式涉及我方财务与合同流程，我需要先与项目、财务、法务团队对齐项目范围与成本结构后，再给您一个正式方案，预计 X 个工作日内反馈。"

## 需交接的评审人

- **财务**：客户信用评估、账期与坏账风险、现金流影响。
- **法务 / 合同**：付款条款、违约责任、第三方采购合同安排。
- **交付负责人**：范围、人力投入估算、现场调研可行性。
- **销售负责人 / 商务上级**：定价与付款条款最终审批。
- **风险评审**（如有）：大额或非标付款条款的最终把关。

## 不确定性标注

- 客户具体规模、信用、议价能力均未提供，本审查基于"信息缺失"的保守判断。
- 项目预估金额未知，因此无法判断是否触发更高层级的审批门槛。
- 上述替代方案的具体比例仅为常见参考，最终条款须经审批确定。
