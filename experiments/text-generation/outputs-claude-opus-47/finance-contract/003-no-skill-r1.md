# 合同让利风险评估与谈判建议

**结论：不可直接答应，亦不建议据此撰写合同补充条款。** 当前让利组合存在多项高风险点，且关键参数（毛利底线、范围边界、审批权限）未确认。

---

## 一、风险识别

### 1. 价格让利风险（七折）
- 标准报价 200,000 元 → 七折后 140,000 元，让利 60,000 元（30%）。
- **毛利底线未知**，无法判断是否击穿成本线或公司最低折扣权限。
- 可能触发**越权审批**风险，需管理层/财务批准。

### 2. "免费二开"风险（高）
- **范围无边界**：未定义二开的功能模块、工作量上限、交付周期。
- 客户可借此持续提需求，形成**无限责任**与项目无法关闭。
- 可能与产品标准化路线冲突，产生长期维护负担。

### 3. "不限次数培训"风险（高）
- **次数、人数、地点、形式（线上/线下）、讲师级别**均未约定。
- 差旅、人力成本不可控；可能被客户用作变相咨询服务。

### 4. "一年内任何问题免费处理"风险（高）
- "任何问题"含义模糊，可能涵盖**非产品缺陷**（误操作、第三方系统、客户环境）。
- 缺少 **SLA（响应时间、解决时间、服务时段）**，售后成本无法预估。

### 5. 组合风险
- 四项让利叠加，相当于在降价 30% 的同时承担**无限范围的交付与服务义务**，整体项目利润和资源占用风险极高。

---

## 二、审批要求（建议在签单前完成）

| 事项 | 审批方 | 需提供材料 |
|---|---|---|
| 折扣超出销售权限 | 销售总监 / 商务负责人 | 成本测算、毛利测算 |
| 价格低于毛利底线 | 财务负责人 / 总经理 | 项目盈亏分析 |
| 二开范围与工作量承诺 | 产品/研发负责人 | 工作量评估 |
| 售后 SLA 与免费服务期 | 交付/客户成功负责人 | 服务成本估算 |
| 合同补充条款 | 法务 | 条款审阅 |

## 三、可谈判的替代表述（待内部审批后再写入合同）

> 以下为**草拟方向**，具体数值需经管理层与财务核定后填入。

### 1. 价格条款（替代"七折"）
"合同总价为人民币 ___ 元（基于标准报价 200,000 元，按 ___ 折优惠，最终金额以管理层书面批准为准）。本优惠为一次性商务让利，不构成后续项目的价格基准。"

> 建议：以阶梯折扣（如分期付款或绑定续约）换取价格让步，避免直接七折。

### 2. 二开条款（替代"免费二开"）
"乙方为甲方提供免费定制开发服务，范围限于：①功能模块：___；②总人天上限：___ 人天；③交付时间窗口：合同签订后 ___ 个月内。超出上述范围的二次开发，按 ___ 元/人天另行计费，并签署变更单后执行。"

### 3. 培训条款（替代"不限次数培训"）
"乙方为甲方提供免费培训服务：①线上培训不限次数，每次不超过 ___ 小时，参训人数不超过 ___ 人；②线下培训 ___ 次，每次 ___ 人天，差旅费由 ___ 方承担；③培训有效期为合同签订后 ___ 个月。超出部分按 ___ 元/场计费。"

### 4. 售后条款（替代"一年内任何问题免费处理"）
"乙方在合同签订后 12 个月内提供免费售后服务，范围限于乙方交付产品本身的缺陷与故障，不含：①非乙方原因（客户操作、第三方系统、网络环境等）导致的问题；②新增需求与功能变更。服务 SLA：
- 响应时间：严重故障 ___ 小时内，一般问题 ___ 个工作日内；
- 服务时段：工作日 9:00–18:00（如需 7×24，另议）；
- 解决方式：远程优先，必要时现场支持（差旅另议）。"

---

## 四、行动建议

1. **暂缓向客户书面承诺**，先内部核算毛利底线与资源边界。
2. 与产品、交付、财务、法务召开 **签单评审会**，确定可让利的红线。
3. 与客户沟通时，将"让利"转换为"**有边界的优惠包**"，用范围换价格、用承诺换续约。
4. 所有补充条款须在获得**管理层书面批准**后方可签署。
