# 财务与合同审查：finance-contract-003

## 一、总体判断

**不可直接答应，亦不可写入合同补充条款。** 当前请求涉及大幅折扣、免费定制开发、无限次培训和无限期免费售后，属于高风险让利组合，且缺少管理层审批和关键边界定义。需先走审批与澄清流程。

## 二、已知事实

- 标准报价：200,000 元。
- 客户要求：七折（即 140,000 元，让利 60,000 元）、免费二开、不限次数培训、一年内任何问题免费处理。
- 当前未确认：毛利底线、二开范围、培训次数与人数、售后 SLA、管理层审批状态。

## 三、财务与合同风险

1. **价格让利风险**：七折让利 30%（6 万元），未对照毛利底线，可能跌破成本线；且未走折扣审批。
2. **免费二开风险**：无范围、无工作量上限、无验收标准，属于无限定制承诺，研发成本不可控。
3. **不限次数培训风险**：未约定次数、人数、地点、讲师成本，存在被无限消耗资源风险。
4. **一年内任何问题免费处理风险**："任何问题"范围过宽，未区分缺陷修复 / 使用咨询 / 新需求 / 第三方系统问题；无 SLA、无免责边界，可能演变为无限期免费服务。
5. **叠加风险**：四项让利叠加将显著拉低项目毛利，甚至导致亏损交付。
6. **审批合规风险**：未经管理层 / 财务 / 法务批准即承诺，事后无法兑现将引发合同纠纷与信任风险。
7. **合同条款风险**：若直接落笔为"补充条款"，将形成对公司单方强约束的最终条款。

## 四、待澄清问题

- 本项目的毛利底线与最低成交价是多少？
- 二开的功能清单、工作量上限、验收标准是什么？
- 培训的次数、单次时长、人数上限、地点、是否含差旅？
- 售后服务范围（缺陷 / 咨询 / 新需求）、响应时效 SLA、免责情形？
- 客户预算与签单紧迫度，是否有可替代的让利组合？
- 是否已获得销售负责人、财务、法务、交付的书面批准？

## 五、所需审批与评审人

- **销售负责人 / 商务总监**：折扣与让利组合审批。
- **财务**：毛利测算与折扣合规审核。
- **交付 / 研发负责人**：二开范围与工作量评估。
- **售后 / 服务负责人**：培训与售后 SLA 边界确认。
- **法务**：合同补充条款措辞与责任边界。
- **管理层**：超出权限的最终决策。

## 六、可谈判的替代表述（仅供内部讨论，非合同条款）

- **折扣方面**：以"阶梯折扣"或"一次性付款折扣"替代直接七折，例如在达到付款节点 / 签约时限 / 案例授权等条件下给予有限折让。
- **二开方面**：以"包含 X 人天定制开发额度，超出部分按标准人天单价结算"替代免费二开，并附《二开需求清单》作为附件。
- **培训方面**：以"包含 N 场标准培训（每场不超过 M 人，地点限于甲方指定城市一处），超出按 X 元/场结算"替代不限次数。
- **售后方面**：以"自验收之日起 12 个月内提供缺陷修复服务，响应时效按 SLA 附表执行；新需求与使用咨询不在免费范围内"替代"任何问题免费处理"。
- **生效条件**：所有让利以"经甲方按期付款且乙方内部审批通过"为前置条件。

## 七、交接建议

1. 暂不向客户做出任何书面承诺，亦不出具补充条款草稿。
2. 由销售牵头，联同财务测算毛利、交付评估二开、服务评估培训与售后边界。
3. 形成让利方案后提交销售负责人 / 管理层审批。
4. 审批通过后，由法务统一起草补充条款，避免销售自行落笔。
