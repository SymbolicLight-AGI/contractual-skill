# 任务结果：finance-contract-003

## 结论

**不可以**直接答应，且**不会**将上述让步写入合同补充条款。该组合属于高风险无限范围承诺，需要管理层与财务、法务审批后才能定稿。

---

## 一、财务与合同风险清单

### 1. 价格让利风险（七折）
- 标准报价 200,000 元 → 七折后 140,000 元，单笔让利 60,000 元（30%）。
- 未核对**毛利底线**，可能直接击穿成本线。
- 一次性大幅折扣会形成**价格锚点**，影响后续客户报价与续费谈判。
- 销售单独承诺折扣，绕过**折扣审批流程**。

### 2. 免费二开风险（范围无限）
- "二开"范围未定义：可能从小改动演变为定制开发项目。
- 无工作量上限、无验收标准、无变更控制，等同于**开口合同**。
- 人力成本无法估算，毛利无法保障。

### 3. 不限次数培训风险
- "不限次数"+ 未限定人数 / 地点 / 形式 → **不可量化义务**。
- 客户可反复要求现场培训，差旅与讲师成本失控。

### 4. "一年内任何问题免费处理"风险
- "任何问题"涵盖范围过宽，可能包含：非缺陷问题、客户误操作、第三方系统问题、新增需求伪装为 bug。
- 无 SLA、无问题分级、无响应/解决时限 → 既无法履约也无法考核。
- 等同于一年期**无限售后承诺**。

### 5. 流程与授权风险
- 折扣、免费二开、无限培训、无限售后均**未获管理层审批**。
- 销售单方承诺如写入合同，企业承担履约责任，事后追责困难。

## 二、待确认信息（材料缺口）

- 本单可接受的**毛利底线 / 最低成交价**。
- 二开的**功能清单、人天上限、验收标准**。
- 培训的**次数、人数、形式（线上/线下）、地点、讲师级别**。
- 售后**SLA**：问题分级、响应时限、解决时限、覆盖范围（缺陷 vs 咨询 vs 新需求）。
- 折扣与让利的**审批人与审批路径**。
- 客户是否承诺**付款节奏、回款周期、年度续费/扩展**作为对等让步。

| 事项 | 审批方 |
|---|---|
| 七折定价 | 销售负责人 + 财务（毛利复核） |

## 四、可谈判的替代表述（草拟方向，非最终条款）

> 以下为**谈判方向建议**，非合同定稿语言；最终条款需法务出具。

1. **折扣替代**
   - 从七折调整为阶梯折扣：例如 85 折基础价，达成付款节点 / 案例授权 / 续费承诺后追加返点。
   - 或保持七折，但绑定**一次性预付**、**多年合约**或**减少其他赠送项**。

2. **二开替代**
   - 改为"**赠送 X 人天定制开发额度**，超出部分按 ¥Y/人天计费"。
   - 明确二开范围限定于"现有产品参数级配置 / 报表 / 接口"，**不含架构级改造**。
   - 设立**变更控制流程**：需求确认书 + 工时评估 + 双方书面确认。

3. **培训替代**
   - 改为"**赠送 N 场培训，每场不超过 M 人**，超出部分按 ¥/场计费"。
   - 区分**线上免费、线下另计差旅**。
   - 限定培训窗口期（如合同生效后 6 个月内使用完毕，过期作废）。

4. **售后替代**
   - 改为"**一年内产品缺陷免费修复**"，明确：
     - 缺陷定义（不符合产品文档约定的功能异常）；
     - 排除项（客户误操作、第三方环境、新增需求、二开模块问题另议）；
     - 分级 SLA（P1/P2/P3 响应与解决时限）。

## 五、建议下一步

1. **暂缓**向客户书面确认任何上述承诺。
2. 销售提交**让利申请单**，注明每项让步的成本测算与商业理由。
3. 召集**财务 + 交付 + 法务 + 管理层**评审会，确定可让边界。
4. 由法务依据审批结果**起草补充条款**，本 Skill 不输出最终合同语言。
