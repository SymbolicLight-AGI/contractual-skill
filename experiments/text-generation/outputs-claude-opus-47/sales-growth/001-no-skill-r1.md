# 销售跟进建议 — sales-growth-001

## 一、客户画像
- **行业与规模**：华东地区离散制造企业（具体规模、营收未知，不确定）。
- **业务场景**：售后服务与企业知识管理并重，存在新员工培训压力。
- **关键干系人**：
  - 信息化负责人：技术守门人，关注集成与安全风险。
  - 售后负责人：业务推动者，态度积极，潜在内部 Champion。
  - 生产计划经理：相关方，立场未明（不确定）。
- **采购阶段判断**：需求识别 / 方案评估早期阶段（决策时间表未明，存在不确定性）。

## 二、明确事实
1. 客户正在评估"企业知识库 + 售后服务 Agent"两类能力。
2. 现状痛点：知识分散于微信群、Excel、老系统，重复问答多。
3. 明确诉求：沉淀标准问答与工单经验，缩短新员工培训周期。
4. 明确约束：对数据安全与权限隔离敏感。
5. 售后负责人态度积极；信息化负责人对集成风险存疑。

## 三、推断机会（含不确定性标注）
- **核心价值切入点**：以"售后知识沉淀 + 新员工赋能"作为首期场景，ROI 可量化（培训周期、重复问答率、平均响应时长）。
- **扩展空间（推断，需验证）**：
  - 工单经验沉淀 → 工单智能分派/辅助回复 Agent。
  - 知识库可向研发、生产计划等部门横向扩展（生产计划经理在场，可能存在跨部门联动潜力，待确认）。
- **Champion 培养**：售后负责人具备成为业务 Champion 的条件。

## 四、风险信号
1. **集成风险顾虑**：信息化负责人对老系统对接存疑，若不解除可能成为否决方。
2. **数据安全敏感**：需提前准备权限模型、私有化部署/数据隔离方案，否则易卡在合规评审。
3. **预算未知**：无法判断方案规模与商务节奏，存在做无效 PoC 的风险。
4. **竞品情况未知**：可能已在比价，存在被作为陪标的风险。
5. **决策链不完整**：未见到分管领导/CIO 或决策审批人，决策路径不清。
6. **生产计划经理立场不明**：可能是观察者，也可能是潜在反对者。

  - 准备集成与安全专项材料（部署模式、权限模型、审计日志、与老系统/微信生态对接路径）。
  - 拟定 PoC 范围与成功指标建议。
- **产品 / 解决方案团队**：
  - 提供离散制造行业售后 Agent 参考方案与 ROI 测算模板。
- **市场团队**：
  - 调取同行业客户案例与证言素材。
- **法务 / 合规**：
