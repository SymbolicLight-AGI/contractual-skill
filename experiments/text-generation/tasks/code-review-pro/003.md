## 用户请求

## 合成代码片段

```python
def load_orders(client, user_id):
    try:
        return client.fetch_orders(user_id)
    except Exception:
        return []
```

## 背景

该函数用于财务对账。上游接口失败时，返回空数组会让对账任务显示“没有订单”。

## 期望输出
